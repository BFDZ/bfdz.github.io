2019 年度数据报告

BFDZ

日志

发布于：2020年2月1日

本来我没有写年度报告的习惯，过年这段时间恰逢国内疫情扩散，只能宅在家里为国家做贡献。各位读者和我一样玩的无聊的话，不妨跟我一起回顾本站去年的一些变化。

一、谷歌分析篇

博客目前使用的是谷歌分析（Google Analytics）来跟踪和统计用户访问数据，因为 uBlokc Origin 等反广告插件会屏蔽谷歌分析的域名，所以谷歌分析的数值会有一定偏差，根据我之前使用谷歌分析反向代理服务器的经验，实际统计数据大约比现在看到的数值高 10%。

首先我们来看网站去年的受众群众概览：

去年全年会话数 197271，用户数 127746。在统计图中有三次明显的访问量增高：

  1. 2019 年 3 月 7 日网页浏览量达到 2940。分析显示流量主要来自豆瓣，因为豆瓣的猫叔Mack分享了我的一篇博文（2019 年十大热门 BT 站），带来了很多新用户。
  2. 2019 年 11 月 15 日网页浏览量达到 1831，用户主要来自搜索引擎以及自然流量。
  3. 2019 年 12 月 23 日网页浏览量达到 4293。在 12 月 19 日晚上，年度热剧《庆余年》全集遭泄露，我在 20 日发布的全集下载链接被谷歌放到了搜索结果首页，从而产生访问量激增。

用户地理位置中 78.08% 位于中国，毕竟本站是中文博客。国内会话数前五的城市分别是北京、上海、深圳、广州、成都。

再来看当今互联网上流行的操作系统。Windows 和 Android 占据了 82.27% 的会话数，剩下的会话数苹果家占了 16 %，Linux 只有可怜的 1%，最后居然还有 0.01% 的人在用我的童年记忆——塞班系统。

2019 年共发布了 20 篇文章，截至本文撰写前，博客文章总共 140 篇。2019 年浏览量前五的文章：

  1. 绅士站整理
  2. 庆余年第一季泄露版全集网盘下载
  3. BT 种子添加有效 tracker 的方法
  4. 2019 年十大热门 BT 站
  5. 2018 年十大热门 BT 站

二、网站架构篇

很长一段时间用的是自己修改过的旧版 material-x 主题，上上个月换了新版 material-x 主题。

最初网站同步部署在 Github Pages 和 Coding Pages 上面，主要访问由 Github Pages 承担。后来 Github Pages 的国内访问速度逐渐变得很慢，主站访问切到了 Coding Pages，然后 2019 年 11 月份 Coding Pages 发生了一次长时间的严重故障，我将网站部署到了三个 vps 上面。当本地有新文章推送到 Github 的博客仓库时，三个不同位置的 vps 会自动拉取更新，同时使用 Perfops 提供的智能 DNS 服务 FlexBalancers，分流全球的用户访问到最近的服务器。

三、工具服务篇

本站提供的简介自动生成工具 PT-Gen ，有越来越多的朋友在使用。2019 年用户数 7,262，会话数 21,973。

PT-Gen 的 API 服务器经过几次迁移以及优化，目前能够保持稳定的运行。距离上一次服务重启，已经持续稳定运行了 61 天。

最后祝大家新春快乐！
